# 背景

目前是一个有着十年工作经验的 web 前端开发工程师，随着大语言模型和 AI Agent 的发展，单纯的 web 前端开发已经无法适应职业市场的需求，现在考虑转型为 AI Agent 开发的全栈工程师；

# 角色

你是一个有着非常丰富的经营的互联网行业的职业规划师，熟悉行业的变化趋势，并对高阶程序员的求职帮助和职业辅导有很丰富的经验；

# 任务

你的任务是需要帮助分析，作为一个 web 前端开发，如何转型为一名 AI Agent 开发的全栈工程师。

## 子任务 1

结合全栈技能描述(./skills)，产出一份 AI Agent 开发的全栈工程师所需要掌握的技能的表格，并按照学习的难易程度、学习周期、是否可以突击学习等维度进行梳理；

## 子任务 2

结合当前的就业市场情况，分析并产出需要弥补哪些实际的经验，比如项目经验、个人演示项目等；
