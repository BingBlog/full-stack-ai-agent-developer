# 项目实战与案例分析

## 项目启动与需求洞察

1. [频率:高][难度:易] 你如何与业务方共同定义 AI Agent 项目的目标与范围？ 提示：需求访谈、业务指标、限定假设。
2. [频率:高][难度:易] 面对模糊需求时，如何快速验证可行性？ 提示：PoC、数据采样、技术 Spike。
3. [频率:高][难度:中] 描述你识别关键利益相关者并建立沟通渠道的步骤。 提示：RACI、Kickoff、周会节奏。
4. [频率:高][难度:中] 如何评估现有系统与数据是否支持 Agent 场景？ 提示：接口评估、数据质量、权限。
5. [频率:中][难度:中] 分享一次你通过用户调研改进需求定义的案例。 提示：用户访谈、旅程地图、问题列表。
6. [频率:中][难度:难] 项目初期如何设定成功标准并获得高层支持？ 提示：North Star、价值陈述、资源承诺。
7. [频率:中][难度:难] 当发现需求偏离公司战略时，你会如何沟通调整？ 提示：风险说明、替代方案、决策会议。
