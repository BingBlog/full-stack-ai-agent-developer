# 安全合规与治理

## 威胁识别与防御

1. [频率:高][难度:易] Prompt 注入的主要类型有哪些？如何设计输入过滤策略？ 提示：越权指令、数据泄露、说明正则/模型审查。
2. [频率:高][难度:易] LLM 越狱攻击常见手法是什么？如何实施多层防护？ 提示：语义扰动、编码绕过、结合 Guardrails。
3. [频率:高][难度:中] 解释模型滥用（Model Abuse）的检测信号，如何实时阻断？ 提示：异常调用频次、敏感词、阈值拦截。
4. [频率:高][难度:中] 如果攻击者试图通过工具调用窃取数据，你如何限制？ 提示：白名单、权限分级、输出脱敏。
5. [频率:中][难度:中] 描述构建红队测试计划的关键要素。 提示：攻防场景、评估指标、自动化脚本。
6. [频率:中][难度:难] 当检测到潜在 Prompt 泄露时，如何快速定位来源？ 提示：日志追踪、请求 ID、用户画像。
7. [频率:中][难度:难] 分享一次你在安全评估中发现高风险用例并修复的案例。 提示：问题发现、修复方案、验证流程。

## 数据保护与隐私

1. [频率:高][难度:易] 数据分级分类的步骤是什么？如何应用到 AI 数据管道？ 提示：识别、分级、访问策略。
2. [频率:高][难度:易] 在训练/推理阶段如何执行数据脱敏？ 提示：字段加密、tokenization、合成数据。
3. [频率:高][难度:中] 说明差分隐私在模型训练中的应用场景与局限。 提示：噪声注入、隐私预算、精度影响。
4. [频率:高][难度:中] 如何设计数据保留策略与自动清理机制？ 提示：生命周期、Retention Policy、审计。
5. [频率:中][难度:中] 对话日志如何在保留可追溯性的同时保护用户隐私？ 提示：部分脱敏、访问控制、加密存储。
6. [频率:中][难度:难] 跨境数据传输需要满足哪些监管要求？ 提示：数据主权、跨境备案、加密通道。
7. [频率:中][难度:难] 描述你在实际项目中与法务/隐私团队合作的经验。 提示：需求澄清、文件准备、风险评估。

## 合规与标准

1. [频率:高][难度:中] SOC2 Type II 审计关注哪些控制面？AI 项目应如何准备？ 提示：安全、可用性、处理过程。
2. [频率:中][难度:中] ISO/IEC 42001 针对 AI 管理体系提出了哪些关键要求？ 提示：风险评估、透明度、治理结构。
3. [频率:中][难度:难] 国内等级保护（等保 2.0）对 AI 系统部署有何特殊要求？ 提示：边界防护、审计、灾备。
4. [频率:中][难度:难] 在受监管行业（金融/医疗）上线 AI 功能需要通过哪些合规流程？ 提示：数据来源、审批、审计。
5. [频率:低][难度:难] 如何记录模型决策链路以满足可解释性法规？ 提示：日志、模型版本、输入输出存档。
6. [频率:低][难度:难] 供应商评估（Third-Party Risk Management）涉及哪些内容？ 提示：安全问卷、渗透测试、SLA。
7. [频率:低][难度:难] 分享一次你带领团队完成合规审核或整改的经验。 提示：差距分析、整改计划、验收。

## 身份与访问管理

1. [频率:高][难度:易] 最小权限原则在 AI 工具调用中如何落地？ 提示：RBAC、ABAC、令牌权限。
2. [频率:中][难度:易] API 密钥管理的生命周期包括哪些步骤？ 提示：生成、分发、轮换、撤销。
3. [频率:中][难度:中] 在 Agent 架构中如何区分用户身份与服务身份？ 提示：用户令牌、服务账号、代办权限。
4. [频率:中][难度:中] 描述零信任（Zero Trust）理念在 AI 服务中的实现方式。 提示：持续验证、上下文感知、微分段。
5. [频率:低][难度:中] 如何审计和追踪高危操作（如导出数据、执行写操作）？ 提示：审计日志、告警、双人审批。
6. [频率:低][难度:难] 联邦身份认证（SSO/SAML/OAuth）接入 AI 平台需要注意什么？ 提示：回调安全、session 管理、多因素认证。
7. [频率:低][难度:难] 分享一次你处理凭证泄露事件的流程。 提示：撤销、影响评估、通知、修复。

## 成本控制与审计

1. [频率:高][难度:易] 模型调用成本为何需要在安全治理中监控？ 提示：滥用检测、预算控制、异常预警。
2. [频率:中][难度:易] 如何设置 Prompt 预算或配额以防止恶意刷量？ 提示：按用户/租户限额、速率限制、审批流程。
3. [频率:中][难度:中] 费用审计报表应包含哪些字段以支持追责？ 提示：调用来源、时间、模型类型、费用。
4. [频率:中][难度:中] 当发现异常消费时，你会如何调查和处理？ 提示：溯源、停用、补偿机制。
5. [频率:低][难度:中] 讨论如何对供应商账单与内部统计进行对账。 提示：API 计量、日志比对、差异分析。
6. [频率:低][难度:难] 成本优化过程中如何平衡安全需求与体验？ 提示：限制力度、白名单、动态阈值。
7. [频率:低][难度:难] 结合实例说明成本分析如何反哺安全规则。 提示：识别可疑调用模式、调整策略。

## 多租户与隔离

1. [频率:高][难度:易] 多租户架构中常见的隔离策略有哪些？ 提示：逻辑隔离、物理隔离、网络隔离。
2. [频率:中][难度:易] 向量数据库如何实现租户级别的访问控制与配额？ 提示：命名空间、标签过滤、限流。
3. [频率:中][难度:中] 当租户需要自定义模型或工具时，如何保证不会影响他人？ 提示：沙箱、插件审核、资源配额。
4. [频率:中][难度:中] 共享 GPU 集群时，如何防止资源窃取或侧信道攻击？ 提示：MIG、调度策略、容器隔离。
5. [频率:低][难度:中] 多租户日志与监控如何分区，既满足安全又便于运营？ 提示：索引分片、标签划分、权限。
6. [频率:低][难度:难] 描述你在 SaaS 环境中处理租户数据边界问题的经验。 提示：合同约束、技术控制、巡检。
7. [频率:低][难度:难] 发生租户间数据泄露时的应急预案是什么？ 提示：隔离租户、通报、补救措施。
